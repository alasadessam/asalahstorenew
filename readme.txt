# 🛍️ قالب متجر أصالة – متجر إلكتروني عربي متكامل

قالب متجر إلكتروني عربي متكامل بتصميم عصري ومتجاوب، مناسب للمتاجر الصغيرة والمتوسطة.  
يحتوي على صفحة رئيسية، صفحة منتج ديناميكية، صفحة من نحن، صفحة تسجيل الدخول، سلة جانبية، نافذة عرض سريع، ونظام إشعارات.

---

## ✨ المميزات

- تصميم عربي بالكامل (RTL)
- متجاوب مع جميع الشاشات (جوال – تابلت – كمبيوتر)
- صفحة رئيسية احترافية
- صفحة منتج ديناميكية (product.html?id=1)
- صفحة من نحن
- صفحة تسجيل الدخول
- سلة جانبية تعمل بالكامل
- نافذة عرض سريع (Quick View)
- نظام إشعارات Toast
- فلترة المنتجات حسب الفئة
- دعم الوضع الليلي (Dark Mode)
- كود نظيف ومنظم (HTML + CSS + JS)
- بدون أي مكتبات خارجية (إلا Font Awesome)
